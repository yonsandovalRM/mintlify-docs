## Accede a Nexden

1. Entra a [app.nexden.cl](https://app.nexden.cl)
2. Inicia sesión con tu email y contraseña (o regístrate si aún no tienes cuenta)

## Configuración inicial

### 1. Crear o unirte a una clínica

Si eres el responsable de la clínica:

- Crea tu clínica desde la configuración
- Completa los datos básicos: nombre, dirección, teléfono, horarios

Si te invitaron a una clínica existente:

- Acepta la invitación que recibiste por email
- Se te asignará un rol (Owner, Profesional o Recepcionista)

### 2. Agregar usuarios (si eres Owner)

- Ve a **Configuración → Usuarios**
- Invita al equipo por email
- Define el rol de cada persona

### 3. Agregar tus primeros pacientes

- Ve a **Pacientes → Crear paciente**
- Ingresa nombre, RUT, teléfono, email y datos necesarios

### 4. Crear tu primera cita

- Ve a **Agenda**
- Haz clic en el día y hora donde quieras agendar
- Selecciona el paciente y el servicio

## Resumen rápido

| Acción             | Dónde encontrarlo   |
| ------------------ | ------------------- |
| Ver la agenda      | Menú lateral → Agenda |
| Crear paciente     | Pacientes → Crear paciente |
| Crear cita         | Agenda → Clic en día/hora |
| Ver ficha clínica  | Pacientes → [Paciente] → Ficha |
| Cobrar / registrar pago | Pacientes → [Paciente] → Pagos |
